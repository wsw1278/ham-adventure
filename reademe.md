# 哈姆大冒险 - 平台跳跃游戏

一个使用Unity引擎开发的2D平台跳跃类游戏。本项目完整展示了从场景搭建、角色控制、交互逻辑到UI设计的游戏开发全流程。

## 🎮 项目亮点与概述

- **完整开发流程**：涵盖起始界面、游戏核心玩法、资源管理及结束界面。
- **细致的交互设计**：包含静态/动态障碍物、移动平台、可收集道具、通关终点等丰富的游戏元素。
- **多感官体验**：为界面、角色动作及关键事件配置了相应的背景音乐与音效。
- **模块化设计**：使用Prefab（预制体）管理可复用资源，并通过Tag和Collider驱动交互逻辑。

## 🕹️ 如何运行游戏

### 方案一：直接游玩（推荐）
1. 前往本仓库的 [Releases] 页面。
2. 下载最新版本的 `HamAdventure_Windows.zip` 压缩包。
3. 解压后，运行 `哈姆大冒险.exe` 即可开始游戏。

### 方案二：在Unity中查看与编辑源码
1. 克隆本仓库到本地。
2. 使用 **Unity 2022.3** 或更高版本打开 `Project` 文件夹。
3. 在项目窗口中打开 `Assets/Scenes` 下的主场景，点击播放按钮进行预览或编辑。

## ✨ 核心功能与设计细节

### 界面设计
- **起始/末尾界面**：
    - 创建了背景音乐 (`Audio Source`) 以烘托氛围。
    - 使用 `Image`, `Text`, `Button` 等UI组件构建界面。
    - 按钮绑定了场景加载函数，实现开始与重启游戏的功能。
- **游戏内UI**：
    - 创建了与玩家脚本绑定的动态文本，用于实时显示分数、生命值等信息。

### 游戏场景与关卡设计
- **多层次背景**：使用 `Tilemap` 创建场景，通过设置不同的 **Layer**、**Order in Layer** 和 **Collider** 来实现视觉层次与物理交互判定。
- **空气墙**：通过不可见的碰撞体防止角色进入未设计区域。
- **资源管理**：将角色、道具、障碍物等制作为 **Prefab（预制体）**，并将重复出现的资源置于空对象下进行集中管理，提升场景编辑效率。

### 可交互对象设计
1.  **障碍物**
    - 设置专属的 **Tag** 和 **Collider** 用于交互判定。
    - 为移动型障碍物创建了 **Animation** 动画，并通过 **Animator Controller** 进行控制。
    - 通过脚本使其在两个空对象（运动端点）之间移动。

2.  **移动平台**
    - 具备两个 **Collider**：一个用于判断角色是否站在平台上以实现跟随移动，另一个结合 **Platform Effector 2D** 组件实现角色的单向站立与穿透（跳跃下落）。
    - 同样具备端点间的移动动画与控制逻辑。

3.  **资源道具**
    - 设置专属的 **Tag** 和 **Collider** 用于收集判定。
    - 添加了旋转、浮动等 **Animation** 动画效果，增强视觉反馈。

4.  **终点**
    - 设置 **Collider** 触发通关判定，实现场景跳转。
    - 添加了 **Audio Source** 和配套脚本，在玩家到达时播放胜利音效。
    - 添加了动画效果增强表现力。

### 玩家角色设计
- **视觉与物理**：配置图像资源、渲染层级 (`Order in Layer`) 和碰撞体 (`Collider`)。
- **相机跟随**：将主摄像机直接置于玩家角色对象内，实现流畅的全程跟随。
- **控制与交互**：
    - 编写角色控制脚本，处理移动、跳跃等操作。
    - 绑定多个 **Audio Source**，在不同交互（如跳跃、收集）时播放对应音效。
- **动画系统**：针对移动、跳跃、静止等不同状态，创建了相应的动画片段，并通过 **Animator Controller** 配合脚本逻辑进行状态切换与控制。

## 🛠️ 技术栈
- **游戏引擎**：Unity
- **编程语言**：C#
- **核心概念应用**：
    - 组件化设计 (`Component`)
    - 预制体 (`Prefab`) 与资源管理
    - 物理系统 (`Collider 2D`, `Rigidbody 2D`)
    - 动画系统 (`Animation`, `Animator Controller`)
    - 音频系统 (`Audio Source`)
    - 用户界面 (`Canvas`, `UI Toolkit`)
    - 2D关卡编辑 (`Tilemap`)
